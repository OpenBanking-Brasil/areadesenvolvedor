v1.0.0-rc6.8 Release Notes

Release Notes para a v1.0.0-rc6.8 do projeto Open Banking Brasil.

Estas mudanças foram aprovadas pelo Comitê de Mudanças do projeto Open Banking Brasil.

Implementações na API - Dados Cadastrais

Mudança Descrição
Todas as Operações. Correção da expressão regular do campo amount

Implementações na API - Cartão de Crédito

Mudança Descrição
Todas as Operações. Correção da expressão regular dos campos e aumento do maxLength para 20 (amount, billTotalAmount, billMinimumAmount, limitAmount, usedAmount, availableAmount e brazilianAmount)

Implementações na API - Contas

Mudança Descrição
Todas as Operações. Correção da expressão regular dos campos e aumento do maxLength para 20 (amount, availableAmount, blockedAmount, automaticallyInvestedAmount, overdraftContractedLimit, overdraftUsedLimit e unarrangedOverdraftAmount)

Implementações na API - Operações de Crédito - Empréstimos

Mudança Descrição
Todas as Operações. Correção da expressão regular dos campos e aumento do maxLength para 20 (amount, contractAmount, feeAmount, chargeAmount, contractOutstandingBalance, paidAmount e warrantyAmount)

Implementações na API - Operações de Crédito - Financiamentos

Mudança Descrição
Todas as Operações. Correção da expressão regular dos campos e aumento do maxLength para 20 (amount, chargeAmount, contractAmount, feeAmount, contractOutstandingBalance, paidAmount e warrantyAmount)

Implementações na API - Operações de Crédito - Adiantamento a Depositantes

Mudança Descrição
Todas as Operações. Correção da expressão regular dos campos e aumento do maxLength para 20 (amount, chargeAmount, feeAmount, contractOutstandingBalance, paidAmount, warrantyAmount e contractAmount)

Implementações na API - Operações de Crédito - Direitos Creditórios Descontados

Mudança Descrição
Todas as Operações. Correção da expressão regular dos campos e aumento do maxLength para 20 (amount, chargeAmount, feeAmount, contractAmount, warrantyAmount, contractOutstandingBalance e paidAmount)
